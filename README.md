# kotlin-lab2

##  Description
Projet Kotlin pour le TP/Lab 2.  
Ce projet affiche les horaires d’ouverture selon le jour et indique si Little Lemon est ouvert ou fermé selon l’heure.

##  Contenu
- Task 1 : Afficher les horaires selon le jour
- Task 2 : Afficher “open now” ou “closed now”
- Task 3 : Bonus (score final)

## Utilisation
Modifier les variables `weekday` et `hour` dans `main.kt` puis exécuter le programme pour tester les résultats.
